Site Info

This is a static website because reasons:

  • Performance: Static websites almost always load faster than apps.
  • Durability: I don't have to maintain frameworks and libraries to make it work because it's based on standard web technology.
  • Flexibility: The simplicity of the build makes it very easy to make changes
  • Portability: Just grab the files and go.

Key Apps

  • 11ty is the site generator.
  • Sublime Text is my code editor.
  • Obsidian is my markdown editor for blog entries and portfolio case studies.
  • Awesome Screenshot is my favorite tool for grabbing screensots and recordings.
  • Amazon S3 & Cloudfront for hosting. I don't love AWS but I do love paying 18¢ / month.

Typography

The primary typeface is Tomato Grotesk by The Designers Foundry.

For extended prose we use a native serif: Hoefler on MacOS & iOS, Palatino on Android and Windows.
